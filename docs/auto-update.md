# 스케쥴 자동 업데이트 전략

## 개요

현재 오버더월 스케쥴은 사용자들이 직접 업데이트하는 방식으로 이루어지고있다. 하지만 매주 초기에는 멤버들이 스케쥴 공지 및 업데이트가 늦어지는 문제가 있다
이를 해결하기 위해 스케쥴 자동 업데이트 기능을 추가하고자 한다.

## 기능 요구사항

1. 자동 업데이트 대상

- 미정, 휴방, 게릴라 상태의 스케쥴

2. 자동 업데이트 주기

- 매일 2시간마다 자동 업데이트(주기는 사용자 설정 가능)

3. 자동 업데이트 방법

- 각 멤버들의 치지직 다시보기 리스트를 가져오는 API를 호출하여 다시보기 데이터를 가져옴
- 만약 멤버의 오늘 스케쥴이 미정, 휴방, 게릴라 상태가 아니라면 자동 업데이트하지 않음
- 만약 멤버의 오늘 스케쥴이 미정, 휴방, 게릴라 상태라면 해당 멤버의 다시보기 또는 라이브 상태를 확인하고 해당 상태로 업데이트함
- 다시보기 또는 라이브 상태의 제목과 시작 시간을 스케쥴 테이블에 업데이트함
